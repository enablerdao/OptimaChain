# OptimaChain UI

このディレクトリには、モダンなフロントエンドフレームワーク（Vite）を使用したOptimaChainのユーザーインターフェースが含まれています。

## 特徴

- 高速な開発環境
- モダンなJavaScriptとCSSのサポート
- 最適化されたビルド出力

## 開発方法

```bash
# 依存関係のインストール
cd vite-app
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## ディレクトリ構造

- `src/` - ソースコード
  - `assets/` - 静的アセット
  - `css/` - スタイルシート
  - `img/` - 画像ファイル
  - `js/` - JavaScriptファイル
- `public/` - 静的ファイル（そのままコピーされるファイル）

## 技術スタック

- Vite - ビルドツール
- Three.js - 3Dグラフィックス
- Chart.js - データ可視化
