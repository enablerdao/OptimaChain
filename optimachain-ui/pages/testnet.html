OVERVIEW

テストネット概要

OptimaChainのテストネットは、メインネットローンチ前に機能とパフォーマンスをテストするための環境です。

OptimaChainのAlphaテストネットでは、以下の革新的な機能を体験することができます：

  • Adaptive Proof-of-Stake (APoS)：ネットワークの状態に応じて自動的にパラメータを調整する革新的なコンセンサスメカニズム
  • 動的シャーディング：ネットワークの負荷に応じて自動的にシャードの数を調整し、スケーラビリティを向上
  • 高速トランザクション処理：最大1,000 TPSの処理能力（テストネット段階）
  • 即時ファイナリティ：1秒未満のファイナリティを実現
  • スマートコントラクト2.0：複数言語対応、並列実行、形式検証機能を備えた次世代スマートコントラクトプラットフォーム

テストネットフェーズ

フェーズ1: 内部テスト

コアチームによる内部テスト（完了）

フェーズ2: アルファテストネット

限定的な外部バリデータによるテスト（現在）

フェーズ3: パブリックテストネット

一般公開テストネット（2024年Q4予定）

フェーズ4: メインネット準備

最終テストとメインネット準備（2025年Q2予定）

PARTICIPATION

テストネットへの参加方法

OptimaChainのテストネットには、バリデータとしての参加とユーザーとしての参加の2つの方法があります。

バリデータとしての参加

バリデータとしてOptimaChainテストネットに参加することで、ネットワークの安全性と分散性に貢献し、 テストトークンの報酬を獲得することができます。また、メインネットローンチ時の優先的なバリデータ選定の対象となります。

必要条件

  • ハードウェア要件：
    • CPU: 4コア以上
    • RAM: 8GB以上
    • ストレージ: 500GB SSD以上
    • ネットワーク: 100Mbps以上の安定した接続
  • ソフトウェア要件：
    • Ubuntu 20.04 LTS以上
    • Docker
    • Docker Compose
  • その他：
    • 24時間365日の運用
    • セキュリティ対策
    • モニタリング

ユーザーとしての参加

ユーザーとしてOptimaChainテストネットに参加することで、ウォレットの使用、トランザクションの送信、 スマートコントラクトの操作など、様々な機能を体験することができます。

参加手順

  1. OptimaWalletをインストール：

    OptimaChainテストネットに接続するためのウォレットをインストールします。

    OptimaWalletをインストール →
  2. テストネットに接続：

    ウォレット設定からテストネットに接続します。

    ネットワーク名: OptimaChain Testnet
    RPC URL: https://testnet.optimachain.io
    チェーンID: 12345
    シンボル: OPT
  3. テストトークンを取得：

    ファウセットからテストトークンを取得します。

    ファウセットにアクセス →
  4. 機能を試す：

    トランザクションの送信、スマートコントラクトの操作、DAppsの利用など、様々な機能を試してみましょう。

FAQ

よくある質問

OptimaChainテストネットに関するよくある質問と回答です。

テストネットのトークンは実際の価値がありますか？

+

いいえ、テストネットのトークン（OPT）は実際の経済的価値を持ちません。テスト目的でのみ使用され、 メインネットローンチ時に実際のトークンと交換されることはありません。

バリデータとして参加するための最小ステーキング量はありますか？

+

テストネットでは、バリデータとして参加するために10,000 OPTのテストトークンが必要です。 これらのトークンはファウセットから取得できます。メインネットでの最小ステーキング量は 後日発表されます。

テストネットはいつまで稼働しますか？

+

現在のアルファテストネットは2024年Q4まで稼働する予定です。その後、パブリックテストネットに 移行し、2025年Q2までテストを継続する計画です。メインネットは2025年Q3にローンチ予定です。

テストネットでスマートコントラクトを開発できますか？

+

はい、テストネットではスマートコントラクトの開発とデプロイが可能です。Rust、Go、JavaScript、 TypeScript、Solidityなど、複数の言語をサポートしています。開発者ドキュメントを参照して、 スマートコントラクト開発の詳細をご確認ください。

テストネットの参加者に報酬はありますか？

+

アルファテストネットの積極的な参加者には、メインネットローンチ時のエアドロップ対象となる 可能性があります。また、バグバウンティプログラムを通じて、重要なバグや脆弱性の報告に対して 報酬を提供しています。詳細は後日発表されます。
